Forklaring: Få commits på GitHub

Vi skal være ærlige — vi har ikke fået brugt GitHub ordentligt under dette
projekt og pga. diverse fejl og problemer.

Vi er fire med hver vores computer og tre forskellige styresystemer. Når
vi prøvede at pushe opstod der hele tiden fejl vi ikke kunne løse enten
i BlueJ som ikke rigtig er bygget til Git eller konflikter fordi Windows
og Linux håndterer linjeskift forskelligt. Vi gav op på det ret hurtigt og
delte filer via USB-stik i stedet.

Det meste af kodningen foregik alligevel ved at vi sad sammen og kodede på
én maskine fra søndag d. 1. juni og frem til aflevering. Inden vi
committede ryddede vi også op i en masse BlueJ-specifikke filer som var
rodet ind i projektet.

Så historikken ser tom ud men arbejdet er lavet — det er bare ikke
registreret undervejs som det burde være.
